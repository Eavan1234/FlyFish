## 教程
1. 登录微信公众平台进入小程序开发，下载开发工具
2. Github中点击Code，打包下载(download zip)
3. 打开本地开发工具导入包
1. 确认project.config.json中的projectname和本地是否一致
2. miniprogram/envList.js中的环境配置改为自己的环境
3. 需要手动创建数据库表：云开发 -> 数据库
